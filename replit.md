# Xara Labs Website

## Overview
Professional corporate website for Xara Labs LLC, a technology consulting company based in San Jose, CA.

## Repository
- GitHub: git@github.com:xaralabs2/xara-labs-website.git

## Tech Stack
- Frontend: React with TypeScript, Vite, Tailwind CSS
- Backend: Express.js (minimal - contact form only)
- Database: PostgreSQL (for contact inquiries)
- Fonts: Inter (clean sans-serif throughout)

## Pages
- Home (/) - Hero, services overview, CTA
- Services (/services) - Detailed service descriptions
- About (/about) - Company background and credibility
- Contact (/contact) - Contact form and information

## Services Offered
- AI & Intelligent Systems
- Custom Software Development
- Data & Platform Engineering
- Technical Strategy & Roadmaps
- Team Augmentation & Embedded Expertise

## Contact Information
- Email: hello@xaralabs.xyz
- Location: San Jose, CA

## Development
Run with: `npm run dev`
Database push: `npm run db:push`

## Design Guidelines
- Clean, minimal, professional visual style
- White background with neutral color palette
- Strong typography using Inter font
- Generous spacing, no heavy animations
